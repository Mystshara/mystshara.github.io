## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/Mystshara/mystshara.github.io.git
cd mystshara.github.io

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Development
```bash
npm run dev    # Start development server
npm run build  # Build for production
npm run preview # Preview production build
```

## 🔧 Customization

### Adding New Services
Edit the `services` array in `assets/App.jsx`:
```javascript
const services = [
    {
        title: 'Your Service',
        description: 'Service description',
        icon: '🎯',
        features: ['Feature 1', 'Feature 2'],
        color: 'linear-gradient(135deg, #color1 0%, #color2 100%)'
    }
];
```

### Adding New Projects
Edit the `projects` array in `assets/App.jsx`:
```javascript
const projects = [
    {
        title: 'Project Name',
        description: 'Project description',
        link: 'https://github.com/...',
        icon: '🛠️',
        category: 'Category'
    }
];
```
